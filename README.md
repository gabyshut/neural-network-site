# Цифровой справочник "Нейросети"

Статический веб-сайт-справочник по теме нейросетей и искусственного интеллекта для образовательных целей.

Сайт доступен по [ссылке](https://gabyshut.github.io/neural-network-site/)

## Описание проекта

Проект представляет собой цифровой справочник, предназначенный для школьников, студентов технических специальностей и специалистов в области искусственного интеллекта. Сайт содержит структурированные статьи по теме нейросетей, иллюстрации, видеоматериалы и глоссарий специфических терминов.

**Технологический стек:**
- **Frontend:** HTML5, CSS3, JavaScript (ES6+)
- **Методология CSS:** БЭМ (Блок-Элемент-Модификатор)
- **Сборка:** Webpack
- **Инструменты разработки:** Visual Studio Code, Figma (дизайн-макет)
- **Контроль версий:** Git

## Выводы и результаты

В ходе проекта были реализованы и протестированы следующие решения:

1. **Статическая архитектура:** Отказ от серверной логики в пользу статического сайта обеспечил высокую скорость загрузки и простоту развертывания.

2. **Методология БЭМ:** Применение БЭМ для организации CSS позволило создать масштабируемую и поддерживаемую систему стилей, избежав проблем с коллизией имен классов.

3. **Modern tooling:** Использование Webpack для сборки проекта упростило управление зависимостями, минификацию кода и применение автопрефиксов, что улучшило производительность сайта.

4. **Полноценный рабочий процесс:** Интеграция Figma для дизайна, VS Code для разработки и Git для контроля версий создала эффективный pipeline от макета до production-развертывания.

Проект подтвердил эффективность подхода "статический сайт + современные инструменты сборки" для образовательных ресурсов, где важны доступность, скорость работы и низкие требования к хостингу.

## Установка и запуск

### Системные требования
- Node.js 14+
- npm или yarn
- Git

### Развёртывание для разработки

```bash

# Установка зависимостей
npm install

# Запуск в режиме разработки с hot-reload
npm run dev

# Сборка для production
npm run build

```
